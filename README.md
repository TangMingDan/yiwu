# yiwu App使用详情
### 下载安装
该软件为android环境下运行的应用，安卓手机直接下载该软件的apk文件安装便可运行
#### 1.进入首页
点击桌面图标后进入首页，点击底部导航栏，我的， 进入该页面
<img src="https://img-blog.csdnimg.cn/20191128211122388.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> >

#### 2.点击立即登录
点击立即登录按钮，进入下面界面
<img src="https://img-blog.csdnimg.cn/20191128211243321.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 

#### 3.注册
由于当前尚无账号，所以点击右上角注册按钮，进入注册页面
<img src="https://img-blog.csdnimg.cn/20191128211404584.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 
输入完整信息，点击获取邮箱验证码，便可以获得验证信息，信息输入完毕之后，便可再次进入登录界面重新登录。

#### 4.首页查看
点击底部导航栏中的主页，便可进入主页查看商品内容
<img src="https://img-blog.csdnimg.cn/20191128211609966.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 

可以点击商品查看商品详情，点击收藏按钮之后便可以成功将该商品收藏
<img src="https://img-blog.csdnimg.cn/20191128211708556.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 
除此之外，可以点击首页的分类，例如我点击图书，便查询该出以下商品
<img src="https://img-blog.csdnimg.cn/20191128211818275.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 


也可以输入查询内容直接进行查询，例如查询输入高数之后，点击搜索或者回车键便可以搜索
<img src="https://img-blog.csdnimg.cn/20191128211937431.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 


#### 5.我的收藏
收藏成功的商品，可以在我的收藏中看见

如果您想要购买该商品，可以点击该商品去联系卖家，与他实时商谈

#### 6.分享界面
除了商品页面之外，你可以底部导航栏中的分享，便可查到推荐的分享内容
<img src="https://img-blog.csdnimg.cn/20191128212316103.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 
你也可以点击右下角的添加按钮，添加自己想要分享的内容
<img src="https://img-blog.csdnimg.cn/20191128212355369.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 

发布成功之后，你可以分享界面中找到自己的分享

<img src="https://img-blog.csdnimg.cn/20191128212422638.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 

#### 7.商品管理
你可以点击首页底部导航栏中的我的，去查看自己的商品信息
<img src="https://img-blog.csdnimg.cn/20191128212757888.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 
例如查看你自己已经分享上架的物品，如果已经卖出，或者不想上架了，你可以点击下架该商品
<img src="https://img-blog.csdnimg.cn/2019112821262426.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3FxXzQyNzgyNzgy#pic_center" width="50%" height="700"/> 

如果你有想要上架卖出的商品，你也可以在点击我也要上架商品按钮去上架商品
